## 🚀 How to Set Up and Launch

### Step 1 — Configure Supabase
1. Create a project at supabase.com
2. Run `SUPABASE_SETUP.sql` in Supabase SQL Editor
3. Go to Project Settings → API
4. Copy your Project URL and anon key
5. Open `src/App.jsx` and paste on lines 8–9:

```js
const SUPABASE_URL = "https://YOUR-PROJECT.supabase.co";
const SUPABASE_KEY = "YOUR-ANON-KEY";
```

### Step 2 — Upload to GitHub
Upload all these files keeping the exact folder structure:
```
plannova-ai/
├── index.html          ← upload to ROOT
├── vite.config.js      ← upload to ROOT
├── package.json        ← upload to ROOT
├── favicon.svg         ← upload to ROOT
├── README.md           ← upload to ROOT
└── src/
    ├── main.jsx        ← upload inside src/ folder
    └── App.jsx         ← upload inside src/ folder (rename from PLANNOVA-AI-Review-Center.jsx)
```

### Step 3 — Deploy on Vercel
1. Go to vercel.com → Add New Project
2. Connect your GitHub repository
3. Click Deploy
4. Your site is live in 2 minutes! ✅
